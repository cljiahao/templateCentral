# NestJS Skills

All skills for NestJS projects scaffolded from templateCentral.

| Skill | Description |
|-------|-------------|
| `scaffold/` | Use when the user wants to start a new NestJS backend project, create a new API, or scaffold a project with modular architecture and Docker support. |
| `code-standards/` | Use when writing or reviewing TypeScript code in a NestJS project — covers naming, decorators, module structure, Zod DTOs, and dependency rules. |
| `add-module/` | Use when the user asks to create a new feature module, add CRUD endpoints, or wire up a new resource in a NestJS project. |
| `add-test/` | Use when adding tests, test coverage is missing for controllers or services, or the user asks to write unit or e2e tests. |
