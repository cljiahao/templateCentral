# FastAPI Skills

All skills for FastAPI projects scaffolded from templateCentral.

| Skill | Description |
|-------|-------------|
| `scaffold/` | Use when the user wants to start a new Python backend project, create a new FastAPI API, or scaffold a project with layered architecture and Docker support. |
| `code-standards/` | Use when writing or reviewing Python code in a FastAPI project — covers naming, types, imports, error handling, and layer dependency rules. |
| `add-endpoint/` | Use when the user asks to create a new API route, add a POST/GET/PUT/DELETE endpoint, or wire up a new resource in a FastAPI project. |
| `add-test/` | Use when adding tests to a FastAPI project, test coverage is missing for endpoints or logic, or the user asks to write unit/integration tests. |
